## Technical Support

### The site is not loading. What should I do?
Try these steps:
1. **Refresh page:** Ctrl+Shift+R (hard refresh)
2. **Clear cache:** Browser settings
3. **Try different browser:** Chrome, Firefox, Safari
4. **Check internet:** Connection stable?
5. **Contact us:** If issue persists

### I can't log in. Help!
Common solutions:
1. **Reset password:** Click "Forgot Password"
2. **Check email:** Correct email address?
3. **Clear cookies:** Browser settings
4. **Disable extensions:** Try incognito mode
5. **Contact support:** Still stuck?

### My poems aren't saving. Why?
Possible reasons:
